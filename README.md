# StarbucksAnalysis
A analytical project for Starbucks dataset


## Index/src
 - csv2xlsx :csv格式转xlsx格式
 - drawChart :画图模块，大部分画图函数请放在这里
 - exceltosql: xlsx转sqlite db格式。
 - models:数据库模型类，使用sqlcodegen 生成的请勿改动
 - query :请在这里编写大部分查询数据的函数
 - main :主模块
 - /dataset :原始数据excel文件
